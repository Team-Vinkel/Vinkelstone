1. How to create a deck

Go to the Cards section

Select the cards you want for your new deck

Go to the Create a deck section

Give the deck a name and select the desired deck class

Click the button "Sumbit"

2. How to create a card

Go to the Create a card section

Select one of the given types of cards: Minion, Spell, Weapon or Hero Power

Fill the forms with the needed Information

Click the button "Create"
